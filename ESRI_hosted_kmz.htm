Here is shown the most of the potential of the web mapping

Here is presented a map, managed by the ESRI Api (Application Programming Interface) system. This is in fact the only one who require an not free account on ESRI. (the only of this demonstration, multiple opensource and or free alternativ will be shown



The map displayed, display a cloud of point. This clound of points are in fact a data file. This showcase us the different cities of this county. However it could be the different monuments, attractions, events, or event the three at the time. This require to have those files created, or to create it.

on the technical side, the file is hosted on an ESRI Server, however it is possible to host it locally, and then call it inside the map, this is done in google maps using kml file.
