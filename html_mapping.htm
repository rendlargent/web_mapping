Simple 'click on map' on Ryde beach

Demonstration of clickable zones (yellow) with three zones types: polygon, circle (technically disk), and rectangle (or square)

The square, directly send to the Ryde Super Bowl website, but as shown in the results pages, it can send to a description pages


This could be used to make some 'more info' maps, to help tourists to help find what they need in precise certain places.

Here, the taken example show the Ryde beach, because it is susceptible that tourists spend time there and search for an ATM,

if it's authorized to walk through the harbor, search for a parking, etc.


on the technical side, this is really the easiest way to do it. However it has multiple cons :

  • the image can't be resized, because the mapping is made with pixels coordinate
  • the can't move inside the 'map', wich is, in fact, a photo
  • this can get really tiedious to display, to surf through and to create if there is a lot of places to map
